## 特性
- [x] 基于Codable的扩展
- [x] JSON映射为object
- [x] object映射为JSON
- [x] 嵌套对象
- [x] 支持nil对象
- [x] 支持默认值
- [x] 支持String和整数、浮点数、Bool之间的转换

## 安装

```ruby
pod 'ObjMapper'
```
